# ERP Widget Project — Enterprise Management System

Full-stack ERP management system built with ds-widget component library.

## Modules
- System administration (users, groups, permissions)
- Organization management
- Multi-language localization
- Email management

## Architecture
- **Frontend**: Next.js 10 + React 17 + ds-widget + Material-UI + AG Grid
- **Backend**: Node.js Express + PostgreSQL
- **Auth**: Token-based with RBAC (group → function → factory)
